# Brent Oil Price Change Point Analysis

## Overview

This project analyzes historical Brent crude oil prices to identify structural changes caused by geopolitical, economic, and policy-related events.

The project applies Bayesian Change Point Detection using PyMC and presents the findings through an interactive Flask and React dashboard.

---

## Project Objectives

- Analyze historical Brent oil prices.
- Detect structural breaks using Bayesian Change Point Analysis.
- Associate detected changes with significant world events.
- Build an interactive dashboard for stakeholders.
- Provide policy and investment insights based on the analysis.
