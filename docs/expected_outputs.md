# Expected Outputs of Change Point Analysis

The Bayesian Change Point Model is expected to produce:

1. Posterior distribution of the change point.

2. Estimated average oil price before the change.

3. Estimated average oil price after the change.

4. Credible intervals describing parameter uncertainty.

5. Probability associated with each estimated parameter.

6. Statistical evidence supporting structural breaks.

These outputs will later be compared with historical geopolitical events to develop hypotheses regarding possible causes of the detected price changes.
