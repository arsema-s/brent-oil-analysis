# Understanding Bayesian Change Point Models

A change point model identifies locations in a time series where the statistical properties of the data change significantly.

For Brent crude oil prices, these changes may correspond to geopolitical conflicts, OPEC policy decisions, financial crises, economic sanctions, or global pandemics.

Instead of assuming that one statistical model describes the entire time series, Bayesian Change Point Analysis divides the series into segments.

Each segment has its own statistical parameters such as the average price.

The Bayesian framework estimates both:

* the probability that a change occurred at each time point
* the uncertainty surrounding the estimated change point

Unlike traditional methods, Bayesian models provide full probability distributions instead of single point estimates.
